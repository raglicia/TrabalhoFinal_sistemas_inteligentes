#  Predição de Doença Cardíaca: Avaliação Comparativa de Algoritmos de Classificação

Este projeto realiza uma análise comparativa entre diferentes algoritmos de **classificação** aplicados à predição de doenças cardíacas.  
O objetivo é identificar quais modelos apresentam melhor desempenho utilizando métricas como **acurácia**, **precisão**, **recall**, **F1-score** e **AUC-ROC**.

A base de dados utilizada inclui informações clínicas de pacientes, permitindo avaliar o potencial de técnicas de **Machine Learning** para auxiliar em diagnósticos médicos.
